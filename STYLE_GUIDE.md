## الباب الأول: بنية الوثيقة الإلزامية

**القاعدة (أ) — ترتيب العناصر**
تلتزم كل وثيقة بالترتيب التالي حرفياً:

١. frontmatter (YAML)
٢. حاقنَا الترويسة (`rh-id` + `rh-class`)
٣. بطاقة ضبط الوثيقة (داخل `<div class="docmeta">`)
٤. الديباجة — blockquote (اختيارية لكن مُوصى بها)
٥. الأبواب والمواد
٦. جدول الاعتماد
٧. سجل التغييرات

**القاعدة (ب) — عدم تكرار العنوان كـ H1**
لا يُبدأ جسم الوثيقة بعنوان `#` يكرّر العنوان الموجود في frontmatter. العنوان يظهر تلقائياً على صفحة الغلاف عبر pandoc؛ كرّره يُنتج عنواناً مضاعفاً في PDF.

**القاعدة (ج) — حاقنَا الترويسة**
يوضعان مباشرةً بعد الـfrontmatter في كل وثيقة دون استثناء:

```html
<div class="rh-id">DOMAIN-TYPE-NNN</div>
<div class="rh-class">التصنيف</div>
```

للوثائق السرّية: `<div class="rh-class sec">سرّي</div>`
للسرّي للغاية: `<div class="rh-class ts">سرّي للغاية</div>`

## الباب الثالث: قواعد الغامق والتنسيق

**القاعدة (و) — الغامق لعناوين المواد فقط (قاعدة حاسمة)**
يُقصر الغامق (`**نصّ**`) حصراً على عناوين المواد والتسميات في **بداية السطر**، ويُمنع الغامق في **منتصف الجملة أو نهايتها**.

السبب التقني: محرّك WeasyPrint قد يفقد المسافة بين الكلمة الغامقة والكلمة التالية في النصوص العربية (مما يُنتج تداخل كلمات في PDF).

✅ **صحيح:**
```
**المادة (١) — البناء على ما انتهى إليه الآخرون**
يلتزم كل عضوٍ بالبناء على المتاح…
```

❌ **خاطئ (يُمنع):**
```
يجب **مراعاة الخصوصية** عند بناء النظام.
```

✅ **البديل:**
```
يجب مراعاة الخصوصية عند بناء النظام (انظر المادة ٤).
```

**القاعدة (ز) — الأرقام**
- النثر العربي: أرقامٌ عربية-هندية (٠١٢٣٤٥٦٧٨٩).
- الإصدارات، التواريخ، معرّفات الوثائق، مفاتيح frontmatter: أرقامٌ لاتينية (0-9).

**القاعدة (ح) — الجداول**
تُستخدم جداول Markdown بالأنابيب في ملفات .md. يجب أن يبقى عمود الوصف (العمود الأول) في جداول `docmeta` بعرض لا يتجاوز ثلاثة أرباع المحتوى لضمان تصيير متوازن في PDF.

**القاعدة (ط) — عدم استخدام HTML المعقّد**
يُسمح فقط بعناصر HTML التالية داخل الوثيقة: `<div>` (للـrh-id وdocmeta)، `<img>` (للشعار عند الحاجة). يُمنع استخدام `<table>` HTML أو `<span>` أو أي CSS مُضمَّن inline.

## الباب الرابع: بنية الأبواب والمواد

**القاعدة (ي) — ترقيم الأبواب**
`## الباب الأول: في …` — يُرقَّم بالعربية-الهندية (الأول، الثاني…) لا بالأرقام.

**القاعدة (ك) — صيغة المادة**
كل مادة فقرةٌ واحدة: التسمية الغامقة في أول السطر يليها النص مباشرةً في نفس الفقرة (سطرٌ فارغ بين المادة والتالية — لا سطر فارغ بين التسمية ونصّها).

```markdown
**المادة (١) — العنوان**
نص المادة هنا. يمكن أن يمتد لعدة أسطر.

**المادة (٢) — العنوان**
نص المادة الثانية.
```

**القاعدة (ل) — التسلسل عند الإضافة**
لا يُعاد ترقيم المواد الموجودة عند إضافة مادةٍ جديدة. تُستخدم صيغة «مكرّر»: المادة (١ مكرر)، المادة (١ مكرر ثانياً).
