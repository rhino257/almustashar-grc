# سجلّ الهندسة (Engineering Journal)

مدخلات سرديّة مؤرّخة تحكي **ماذا أنجزنا، متى، ولماذا** — بتفصيل يكفي لأي فريق
استلام أو محادثة جديدة لفهم الرحلة كاملة.

## القواعد
- **Append-only**: لا تُعدَّل المدخلات السابقة. الخطأ يُصحَّح بمدخلة جديدة.
- تسمية الملف: `YYYY-MM-DD-<موضوع-مختصر>.md`.
- استخدم القالب: `templates/journal-entry-template.md`.
- القرارات الجوهرية تُرفَع إلى `docs/adr/` كـ ADR مستقلة، ويُشار إليها من هنا.
