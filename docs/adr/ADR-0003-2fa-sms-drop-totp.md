# ADR-0003: التحقق الثنائي عبر SMS وإسقاط TOTP

- **الحالة:** Accepted
- **التاريخ:** 2026-07-04
- **صلة:** ADR-0002، NIST SP 800-63B-4

## السياق
أردنا تحقّقاً ثنائياً اختيارياً. TOTP (تطبيقات المصادقة) أقوى نظرياً، لكن
جمهورنا غير معتاد عليها، والاعتماد على الهاتف أعلى. NIST SP 800-63B-4 يصنّف
الرسائل قناةً «مقيّدة» (restricted) لا محظورة.

## القرار
- **2FA اختياري عبر SMS** (`users.mfa_enabled`, `otp_codes.purpose='mfa_login'`).
- **إسقاط TOTP** في هذه المرحلة (يبقى قابلاً لإعادة النظر مستقبلاً).

## النتائج
- (+) تجربة مألوفة للمستخدم اليمني وبدون تطبيق إضافي.
- (−) SMS «مقيّدة» أمنياً (تبديل SIM، اعتراض) — مقبولة كمقايضة سياقية.

## البدائل المرفوضة
- **TOTP إجباري/اختياري الآن:** حاجز اعتماد مرتفع للجمهور المستهدف.
